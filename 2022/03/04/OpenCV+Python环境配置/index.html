最简单的OpenCV+Python环境配置

发表于2022-03-04|更新于2022-03-04
|阅读量:

OpenCV+Python环境配置

这是我的OpenCV学习历程，现在写下文档，记录学习中的注意事项

准备材料

anaconda：用于python环境配置和库加载

pycharm：用于编译代码

Anaconda配置

打开Anaconda Prompt

①在命令行输入cd .anaconda

②dir 显示所有文件夹

③cd navigator

④输入python检测是否安装成功，输入exit（）退出

⑤ dir 显示文件

⑥使用 cd scipts 打开文件夹

⑦使用 pip list 查看已安装的包

我们需要的是opencv-python和opencv-contrib-python

⑧使用 pip install opencv-python

pip install opencv-contrib- opencv 下载最新版本的opencv

如果下载慢请换软件源

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python

⑨打开python 输入import cv2 检查是否成功安装,直接返回则为安装成功

⑩输入 cv2.__version__ 查看版本号

以上python环境配置成功

pycharm配置

打开IDE

点击文件中的设置

在python解释器中选择 Python3.9（anaconda3的库）

点击确定即为配置成功

以上完成后即可使用pycharm进行opencv的开发啦
