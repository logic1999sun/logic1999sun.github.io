HEXO+github博客搭建

准备材料

1.安装Git

2.安装Node.js

3.安装Hexo

4.Github创建个人仓库

5.生成SSH添加到Github

6.设置个人域名

8.发布文章

1.安装Git

用于分布式管理你的系统，非常强大的工具。具体学习请到：Git教程

Windows：使用官网下载

linux：只需要在目标文件夹内使用命令：sudo apt-get install git

同时可以使用:git --version

2.安装nodejs

Windows：选择LTS就可以

Linux：

sudo apt-get install nodejs

sudo apt-get install npm

检测版本使用：

node -v

npm -v

3.安装HEXO

配置好Git和nodejs后，创建一个文件夹（Windows下在文件夹右键git bash即可使用命令行）

安装命令：npm install -g hexo-cli

版本： hexo -v

初始化HEXO：hexo init yourname

进入文件夹： npm install

新建完成后，指定文件夹目录下有：

  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • ** _config.yml: 博客的配置文件**

使用 hexo g生成静态网页

使用 hexo server打开本地端口映射网页

在浏览器输入 localhost:4000 即可打开

使用 Ctrl+c 即可关闭服务

4.创建GitHub个人仓库

首先注册GitHub账户，然后点击 New repository

其中特别注意 Repository name 的填写格式，也就是 yourname.github.io

这样才会被GitHub的pages服务识别到

点击create repository即可完成创建

5.生成SSH添加到GitHub

打开 git bash

这里的yourname输入你的GitHub用户名，youremail输入你GitHub的邮箱。这样GitHub才能知道你是不是对应它的账户。

可以用以下两条，检查一下你有没有输对

然后创建SSH,一路回车

这个时候它会告诉你已经生成了.ssh的文件夹。在你的电脑中找到这个文件夹。

一般地址为 C:\Users\yourcomputername.ssh

回到GitHub，打开setting，找到SSH keys的设置选项，点击New SSH key 把你的id_rsa.pub里面的信息复制进去。

然后在 git bash 中检查是否成功

ssh -T git@github.com

6.将hexo部署到GitHub

打开站点配置文件 _config.yml，翻到最后，修改为 YourgithubName就是你的GitHub账户

这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。

然后

其中 hexo clean清除了你之前生成的东西，也可以不加。 hexo generate 顾名思义，生成静态文章，可以用 hexo g缩写 hexo deploy 部署文章，可以用hexo d缩写

注意deploy时可能要你输入username和password。

然后你就可以在这个网址查看你的博客啦！http://yourname.github.io
